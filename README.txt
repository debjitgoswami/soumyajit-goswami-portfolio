### ğŸ¯ Customization Guide:

#### 1. **Replace Placeholder Images:**
   - Project thumbnails: Replace emoji in `.project-thumbnail`
   - Gallery images: Replace content in `.placeholder-image`
   - Profile photo: Replace content in `.profile-placeholder`

#### 2. **Add Your Content:**
   - Edit text in HTML files
   - Update project titles, descriptions, and tech tags
   - Modify social media links in contact.html

#### 3. **Add Videos:**
   In `project-detail.html`, replace the video placeholder:
   ```html
   <video width="100%" controls>
       <source src="your-video.mp4" type="video/mp4">
   </video>
   ```

#### 4. **Change Colors:**
   In `style.css`, modify these variables:
   - Primary background: `#0a0e1a`
   - Accent color: `#00d4ff`
   - Text colors: `#ffffff`, `#8892b0`

#### 5. **Add Real Images:**
   Replace placeholders with:
   ```html
   <img src="path/to/your/image.jpg" alt="Description">
   ```
